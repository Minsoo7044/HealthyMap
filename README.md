# 헬맵
## 카카오맵을 활용하여 실외운동시설을 보여주는 웹앱


### 주요기능

  #### 1. 시설 확인
   - 카카오 api를 사용하여 사용자 주변 실외운동시설을 마커를 통해 보여줌
  #### 2. 기구 확인
   - 시설 상세를 클릭하여 그 시설에 존재하는 운동기구들을 볼 수 있음
  #### 3. 즐겨찾기
   - 자신이 자주 이용하는 시설을 즐겨찾기하여 나의 즐겨찾기를 통해서 볼 수 있음
  #### 4. 공지사항
   - 시설의 변경사항이나 공지사항들을 확인할 수 있음
